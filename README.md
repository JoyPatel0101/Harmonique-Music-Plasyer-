# Harmonique 🎵

**Harmonique** is a sleek and dynamic **music streaming web application** built using **JavaScript** and **Node.js**. It allows users to explore, stream, and manage music with an intuitive interface and seamless backend integration.

## 🚀 Features

- 🎧 Stream music directly from the web interface  
- 📁 Manage your music collection  
- 🔍 Search and filter songs  
- 🖥️ Responsive design for desktop and mobile  
- 💡 Simple and user-friendly interface

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Node.js, Express.js  
- **Database (Optional):** MongoDB / File-based storage (if implemented)  
- **Other Tools:** EJS (for templating), Bootstrap (optional for styling)

## 📂 Project Structure

```
/Harmonique
│
├── public/          # Static files (CSS, JS, Images)
├── views/           # EJS templates for UI rendering
├── routes/          # Route handlers
├── app.js           # Main server file
└── package.json     # Node.js dependencies
```

## 💻 How to Run

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/harmonique.git
   ```

2. Navigate to the project folder:
   ```
   cd harmonique
   ```

3. Install dependencies:
   ```
   npm install
   ```

4. Start the server:
   ```
   node app.js
   ```

5. Open in your browser:
   ```
   http://localhost:3000
   ```
